Introduction

Le pentesting ou tests de pénétration est une étape importante dans la sécurisation d’une application web. Elle consiste à effectuer des tests d’intrusion pour détecter des vulnérabilités web.

Cette étape intervient dans la dernière phase du cycle de développement d’une application après la revue de code statique. De nombreux logiciels d’automatisation de tests de pénétration existent. L’OWASP tient à jour une page qui liste les logiciels de scan web les plus utilisés. Le WASC a également sa propre liste de logiciels de scan web mais la page n’a pas été mise à jour depuis 2010.

Dans ce cours, nous nous intéresserons au logiciel ZAP (Zed Attack Proxy) développé par l’OWASP. Ce logiciel agit comme un proxy. Il se situe entre le navigateur de l’utilisateur et le web et permet d’intercepter toutes les requêtes http. Ce logiciel est disponible pour Windows, Mac et Linux.

Configuration du logiciel ZAP (Zed Attack Proxy)

Lien de téléchargement :
https://www.owasp.org/index.php/OWASP_Zed_Attack_Proxy_Project

Configuration des réglages :

1. Ouvrez votre navigateur préféré (Chrome) et ouvrez les préférences du proxy local :

Désactivez « utilisez un script de configuration automatique ».
Tapez :
IP : 127.0.0.1
Port : 8085

2. Ouvrez le logiciel ZAP :
a. Configuration du proxy local :

Allez dans options puis « Local Proxies » 
Tapez :
Adresse : localhost
Port : 8085

b. Configuration du proxy sortant :

Enfin, toujours dans les options de ZAP allez dans « Connexion » et cochez la case « Utilisez un serveur proxy sortant ».

Tapez :
Adresse : 10.1.4.2 (adresse actuelle du Proxy, pour la trouver, téléchargez le script de configuration automatique du proxy disponible à cette adresse : http://wpad.solutec.lan:8080/wpad.dat)
Port : 8080

Dans la case « Authentification Proxy », cochez « Le serveur proxy sortant requiert une authentification ».
Renseignez votre nom d’utilisateur et mot de passe SOLUTEC et cliquez sur OK.

3. Générer un certificat

Dans les options de ZAP, allez dans « Certificats SSL dynamique »
Cliquez sur « Générer » puis sur « Sauvegarder ».

Allez dans les paramètres de votre navigateur préféré (Chrome) puis dans « Gérer les certificats ».
Dans la nouvelle fenêtre, cliquez sur « Importer », sélectionnez le certificat préalablement enregistré et importez-le dans la catégorie : « Autorités de certification racines de confiance ».

ZAP est maintenant prêt à être utilisé sur votre machine.
